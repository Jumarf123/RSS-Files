# Roadmap

## Near Term

- improve fragmented file recovery planning for carved artifacts
- add structured preview support for CAB/RAR/7Z/ISO beyond header-aware summaries
- expand signature evidence with countersignature/timestamp details
- add settings/about screens in the frontend

## Medium Term

- exFAT deleted-entry support
- image-file input mode
- case timeline and operator annotations
- richer hash and signature evidence panel
- benchmark fixtures for reproducible fast/deep SLA measurement

## Installer / Ops

- cleaner release packaging scripts
- manifest and icon validation in CI
- installer smoke tests in CI
- optional NSIS shortcut-page customization beyond the default Tauri flow
- WebView2 offline installer regression coverage in CI

## Long Term

- optional helper sidecar integration with explicit compliance packaging
- more advanced fragment reconstruction
- threat-intel enrichment for recovered binaries
- deeper case management and chain-of-custody tooling
